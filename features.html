General OPTIMUS features

general release date: 2025.7.10

OPTIMUS has a wealth of features to enhance judgement productivity. Features of OPTIMUS are as follows:

  • Публикация результатов на сайт
  • Смотри любые форматы видео
  • Две методики оценки
  • Плати за то что используешь
  • Работай на понятном языке
  • Доступное оборудованию
  • Работает на одном или по сети
  • Редактор документов
  • Расширение divepool
  • Современный дизайн
  • Индивидуальная тренировка и стажеровка
  • New refactoring code
  • New development tools

#1 Публикация результатов на сайт

Publish results to Web

отображение соревнований in infinity list; поиск мероприятий по фильтрам; автоматическая выгрузка данных:протокол соревнований, фото и состав команды, судейские записки и видео выступлений.
страница результатов - это одностраничный сайт для информационной поддержки соревнований + протокол результатов.

#2 Создавай свои формы документов

Встроенный редактор печатных форм

Комплект поставки включает стандартные формы, но их можно исправлять и создавать новые. Включая формы для web-документо.
Поддерживается множество форматов экспорта докуметов: .PDF; .DOC; .HTML; .JPEG; .BMP.

#3 Смотри любые форматы видео

проигрывает множество видео

В системе используется библиотека FFMPEG для проигрывания видео форматов .AVI; .MP4; .MPEG; .MPG; .MOV; .MKV и др.;

#4 Плати за то что используешь

Доступная стоимость

online конструктор лицензий - Выбирайте дисциплины по которым планируется проведение соревнований. Бесплатная download page.

#5 Работай на понятном языке

Multilingual architecture

Выбор языка ведения справочников (ru/en/uk/fr/es/de); English/Rusian user interface; печатные формы на рус/eng/укр. Детальная документация на русском.

#6 Индивидуальная тренировка и стажеровка на соревновании

Стажеры могут участвовать в оценке выступлений наравне с судьями, при этом их оценки не учитываются при подсчете результата.
Для индивидуальной тренировки (в однопользовательском режиме) доступны все поддерживаемые дисциплины.

#7 Работает на одном или по сети

SQL DB engine

Native support for for all major operating systems, including Windows, Linux and MacOS; Embedded version; экономия трафика сжатием BLOB-полей на клиенте.

#8 Расширение divepool

более 30 готовых набора

В справочнике кроме стандартных наборов ISC собраны наборы федераций федераций DFV, AXIS, USPA, ФПСУ, ФПСР и Axis.

#9 Две методики оценки

одновременно или Ferret

Можно оценивать выступления группой или индивидуально. Что дает необычайную гибкость для разных форматов проведения соревнований.

#10 Доступное оборудованию

работает с различным набором аппаратуры

Применятся исключительно стандартное компьютерное оборудование, которое можно купить в любом магазине: клавиатуры, мониторы, USB. Multiscreen систему можно собрать во множестве вариантов, приемлемой для Вас по бюджету и доступному оборудованию.

#11 Современный дизайн

оформление в стиле Windows 10

Сменяемые цветовые схемы интерфейса; унифицированные экранные формы; New SplashScreen logo and toolbar icons.

#12 New refactoring code

Bug fixes and Quality improvements

Онлайн проверка доступности новой версии; новый формат файла Licence v.4 (crypted); обмен данным через internеt по OpenSSL.

#13 New development tools

Compiled with Delphi 10.4 Sydney; Unicode compatibility added; Uses latest components version;

Fully functional, Free download, Personal licence.
