🟪Technical details

We use Delphi to develop OPTIMUS. This is a

  1. Based on SQL DB engine: SQL-server Native support for all major operating systems, including Windows, Linux and MacOS; Portable (Embedded mode) and standalone installation
  2. Multilingual architecture: Modern English/Rusian user interface in Windows 10 style, rus/eng/ukr language document templates
  3. Built-in editor: Настраивай шаблоны документов с помощью встроенного редактора печатных форм. Predefined set of documents на рус/eng/укр, включая формы для web-документов. Export to multiple formats (PDF, DOC, HTML, JPEG, BMP)
  4. Versatility player: Used modern FFMPEG library to play multiple video format (.MP4;.MPEG;.AVI;.MKV;...)

The OPTIMUS version consists of four numbers: major.minor.patch.build. They differ in several ways:

  • build - compilation number during development;
  • patch - incremental release number is made fixes or improvements;
  • minor - released new functionality that does not affect the old one;
  • major - a release is issued that changes the way old functionality is used.
The version number can be checked in the exefile properties on the Details tab or in application menu About.
