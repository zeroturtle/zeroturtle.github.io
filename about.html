About

What is OPTIMUS?

OPTIMUS is a all-in-one software suite разработанный для оценки outdoor дисциплин Formation Skydiving, Artistic Events, Canopy Formation и подсчет результатов для Wingsuit Flying Performance. Реализует оценку по правилам ISC, легко адаптируется для правил национальных федераций. Обеспечивает создание video replay, трансляцию на DZTV, публикацию результатов на Web-сайт.
Предоставляет доступное low cost решение для организаторов соревнований по парашютному спорту.

Use Cases and Work Flow

Архитектура системы

Для работы OPTIMUS собирается мультискрин система, может реализовываться во множестве вариантов и стоимость может меняться. Полная конфигурация системы состоит из 1 Scoring laptop/PC, и комплект для судейских консолей (цифровая USB-клавиатура и монитор), USB-hub, ТВ и лазерный принтер. На соревнованиях дополнительно понадобятся подключить компьютеры для Vanguard (dubbing station ) и Zodiak (создание Replay-видео).

Аналоги

На сегодня существует несколько альтернативных судейских систем для оценки парашютных дисциплин: InTime Scoring, Omniskore!HD, DynamR.

В каждой системе использованы разные технические решения, отличаются подходом в организации судейских консолей. Все системы работают с цифровым видео. Имеют различные требования для оборудования и набора ПО.
Системы могут масштабироваться под размеры соревнований за счет функциональных модулей, разнесение функций по разным компьютерам подключенных по локальной сети.

Why would I use OPTIMUS?

  1. Для работы системы OPTIMUS и OSHD достаточно одного scoring computer, к которому подключаются все судейские консоли.
  2. Для Intime требуется в 5 раз больше оборудования, по сравнению с системами OSHD и OPTIMUS, т.к. каждая судейская консоль InTime запускается на отдельном компьютере.
  3. Консоли OSHD комплектуются изделием Matrix с убогим LCD-screen, что не сравнится с богатыми возможностями консолей OPTIMUS.
  4. Стоимость лицензии OPTIMUS определяется набором активных дисциплин и предоставляет значительные скидки за счет программ лояльности, в то время как лицензия InTime обходится в 10 раз дороже.
  5. OPTIMUS распространяется бесплатно для индивидуального пользования, также как и OSHD. У InTime однопользовательскую лицензию нужно покупать.
  6. В индивидуальной подписке доступны все поддерживаемые дисциплины, InTime express предлагает отдельные версии для некоторых дисциплин.
  7. В OPTIMUS результаты соревнований можно вставлять в любой сайт, InTime публикует только на своем портале. OSHD не имеет сайта вовсе.
  8. Все системы предлагают свои уникальные решения для оценки выступлений:
    • InTime Online - cloud-based system operates via internet browsers; Экономия организаторам на проезде и проживании судей, оплачивается по $3 за оценку каждого видео. Лишает судей возможности путешествовать, личной встречи и обмена опытом, получать удовольствие командной работы.
    • OSHD может работать с клавиатурами как консоли, но корректировка оценок происходит поочереди на одном экране; Экономия на дурацких Matrix девайсах. Можно применить для единичной оценки выступления, но для соревнований совершенно непригодно.
    • OPTIMUS с модулем Ferret позволяет оценивать выступления автономно и несинхронно
