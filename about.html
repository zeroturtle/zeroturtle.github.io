General OPTIMUS features

Zero-administration approach

No installation (plug-and-play).

Built-in editor

Настраивай свои шаблоны документов с помощью встроенного редактора печатных форм. Позволяет исправлять существующие и создавать новые виды документов, включая формы для web-документов.

Pay-as-you-go licencing

online конструктор лицензий - Выбирайте дисциплины по которым планируется проведение соревнований; Free download.

Расширение divepool

В справочнике содержится more then 30 divepool images, кроме стандартных наборов ISC собраны наборы федераций федераций DFV, AXIS, USPA, ФПСУ, ФПСР и Axis.

Based on SQL DB engine

SQL-server Native support for all major operating systems, including Windows, Linux and MacOS; Portable (Embedded mode) and standalone installation.

Шаблоны документов

Поставляется predefined set of documents на рус/eng/укр; Export to multiple formats (.PDF;.DOC;.HTML;.JPEG;.BMP)

Personal licence

Free for individual use, fully functional, No advertising nor nag-window

Работай на понятном языке

Понятный English/Rusian user interface; Multilingual architecture справочников (ru/en/uk/fr/es/de);

Смотри любые форматы видео

В системе используется библиотека FFMPEG to play multiple video format (.MP4;.MPEG;.AVI;.MKV;...).

Modern design

Оформление в стиле Windows 10; Сменяемые цветовые схемы интерфейса;

Поддержка

Unlimited e-mail support; Онлайн проверка доступности новой версии; Bug fixes and Quality improvements; Free updates during subscription period.

No specialised equipment required

Сборка multiscreen по доступному оборудованию и приемлемому для Вас по бюджету

Две методики оценки

Позволяет оценивать выступления группой (одновременно) или индивидуально (модуль Ferret). Дает гибкость выбора для разных форматов проведения соревнований. Individual or consolidated judgement;

New development tools

Compiled with Delphi 10.4 Sydney; Unicode compatibility added; Uses latest components version; Detailed documentation;

Индивидуальная тренировка и стажеровка на соревновании

Стажеры могут участвовать в оценке выступлений наравне с судьями (judge-intern involved); Для индивидуальной тренировки (в однопользовательском режиме) доступны все поддерживаемые дисциплины.
